1. Install dependencies from executables, if you haven't already
2. Execute DHS_InstallOpenCV.bat
3. Follow instructions from terminal to finalize

If you run into issues, refer to this link https://www.learnopencv.com/install-opencv-4-on-windows/
The script DHS_InstallOpenCV.bat is supposed to automate what this link tells you to do. It is hardcoded to expect the installation to go to C:\dependencies\Win64\Opencv\opencv-master\Installation\x64\vc15\staticlib

4. Restart computer
5. Run the unreal software
